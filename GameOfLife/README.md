# Game of Life

This is an implementation of Conway's Game of Life model. It is visualized in
a canvas. To start running the model, press enter.

## Requirements

- C++ v14 or higher.
- [DeSiGNAR](https://github.com/R3mmurd/DeSiGNAR) library.
- Qt5
